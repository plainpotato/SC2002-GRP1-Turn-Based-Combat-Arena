## Project Overview
A command-line turn-based combat arena game built in Java using OODP and SOLID principles.

## Features
- Warrior and Wizard player classes
- Goblin and Wolf enemy types
- Actions: Basic Attack, Defend, Item, Special Skill
- Status effects: Stun, Defend, Smoke Bomb Invulnerability, Arcane Blast buff
- Multiple difficulty levels with backup spawning
- Extensible architecture using interfaces/strategies

## Tech Stack
- Java
- Maven
- GitHub

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA / Eclipse
3. Run `App.java`

## Branching Strategy
- `main`: stable
- `dev`: integration
- `feature/...`: individual tasks
